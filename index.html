Toast

짧은 피드백 메시지를 화면 하단에 표시합니다. 자동으로 사라지며, 선택적으로 액션 버튼을 제공합니다.

상품이 찜 목록에 추가되었습니다
거래가 완료되었습니다
오류가 발생했습니다. 다시 시도해주세요
인터넷 연결이 불안정합니다 
/* Default Toast */
background:    var(--surface-vuddy-primary);  /* gray-900 */
color:         var(--text-white-primary);
border-radius: var(--radius-large);
box-shadow:    var(--shadow-higher);

/* Action button color */
color: var(--text-vuddy-highlight);           /* yellowgreen-300 */
Components

Tip / Caution

콘텐츠 내부에 인라인으로 삽입되는 안내·경고 메시지입니다.

안내: 거래 전 상대방의 프로필을 확인하는 것을 권장합니다.

성공: 인증이 완료되었습니다. 이제 거래를 시작할 수 있습니다.

오류: 해당 상품은 이미 판매 완료된 상품입니다.

주의: 직거래 시 안전한 장소를 선택하세요.
